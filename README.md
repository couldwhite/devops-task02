## Разделы обучения
1. GIT

Основые команды:
|Команда|Описание|
|-------|--------|
|git add|Добавляет содержимое рабочего каталога в индекс для последующего коммита.|
|git status|Показывает состояния файлов в рабочем каталоге и индексе|
|git commit|Берёт все данные, добавленные в индекс с помощью git add, и сохраняет их слепок во внутренней базе данных, а затем сдвигает указатель текущей ветки на этот слепок|
|git push|Используется для выгрузки содержимого локального репозитория в удаленный репозиторий|
2. TERRAFORM

Чтобы добавить необходимые команды в ваш профиль оболочки,выполните следующую команду:
```
terraform -install-autocomplete
```

[Подробнее смотрите тут](https://runebook.dev/ru/docs/terraform/commands/index)
